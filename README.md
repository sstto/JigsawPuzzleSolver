# Automatic JigsawPuzzleSolver

## Requirements

- Linux or Macos
- python==3.6
- numpy==1.15.0
- scipy==1.0.0
- opencv_python==3.3.0.10
- scikit_image==0.13.1
- matplotlib==2.1.0
- Pillow
- scikit_learn>=0.18.1

we recommend setting up a python 3.6 virtual environment and then using "pip install -r requirements.txt".
